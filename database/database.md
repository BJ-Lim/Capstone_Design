### 학생정보 table
- 학생(학번,이름)
- 주키:학번

### 교수정보 table
- 교수(사번,이름)
- 주키:사번

### 과목정보 table
- 과목(과목코드,분반,과목명,강의자명,학기,학점)

### 서버 table

- 외래키:학번,과목정보,과제코드

### 과제 table

- 외래키: 과목코드,분반
